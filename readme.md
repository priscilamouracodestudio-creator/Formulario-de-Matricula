## 🧠 Desafios e Aprendizados

Durante o desenvolvimento, foquei em resolver problemas reais de interface, como:

### 1. Custom Checkbox (A "Bruxaria" do CSS)
Implementei uma técnica de **Positioning (Relative/Absolute)** para esconder o input padrão e criar uma caixa personalizada que aceita imagens e transições suaves.
* **Aprendizado:** Uso do seletor `:checked` e `&:has()` para manipulação de estados sem JavaScript.

### 2. Flexbox Avançado
Utilizei a propriedade `flex: 0 0 1.5rem` para garantir que elementos de ícones nunca fossem esmagados pelo texto lateral, mantendo a integridade do design em qualquer resolução.

### 3. Hierarquia de Ações
Organizei os botões de **Salvar** (Secundário) e **Matricular** (Primário) seguindo padrões de leitura ocidental e UX, garantindo que o usuário conclua o fluxo de forma intuitiva.
